# TrackAll - Expense Tracker App

TrackAll is a modern, dark-themed Android application designed to help users track their daily expenses efficiently. Built with Kotlin and the MVVM architecture, it offers a seamless user experience with features like category management, monthly summaries, and intuitive data visualization.

## üì± Features

-   **Expense Tracking:** Add, edit, and delete daily expenses with ease.
-   **Category Management:** Categorize expenses (Food, Transport, Entertainment, Other) using intuitive icons.
-   **Monthly Summary:** View your total expenses for the current month at a glance.
-   **Weekly View:** Navigate through the week to see expenses for specific days.
-   **Dark Theme:** A sleek, global dark theme for comfortable usage in any lighting condition.
-   **Swipe to Delete:** Quickly remove expenses with a swipe gesture (includes Undo functionality).
-   **Secure Data:** Local database storage using Room for privacy and offline access.
-   **User Authentication:** Secure login and signup flow.

## üöÄ Getting Started

### Prerequisites

-   Android Studio Iguana or later
-   JDK 17 or later
-   Android SDK API Level 24 (Min) to 34 (Target)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kraonix/TrackAll.git
    ```
2.  **Open in Android Studio:**
    -   Launch Android Studio.
    -   Select "Open" and navigate to the cloned directory.
3.  **Build the project:**
    -   Let Gradle sync and download dependencies.
    -   Click on `Build > Make Project`.
4.  **Run the app:**
    -   Connect an Android device or start an emulator.
    -   Click the "Run" button (Shift + F10).
